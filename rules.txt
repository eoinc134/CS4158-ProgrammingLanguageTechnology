Language Syntax:
- Not case sensitive
- START. ——> END.
- Each line ends with full stop
- Program split into declarations, assignments, inputs and outputs

***Declarations***:
- After START.
- *Capacity String:* SSSS {4 digit integer}
- *Variable Names:*
    - Letters, digits, hyphens
    - Must start with character/underscore
    - …followed by a character
    - Cannot be a series of S’s

→ MAIN. declaration here

***Assignments:***
- MOVE identifier TO identifier
- MOVE integer TO identifier
- ADD identifier TO identifier
- ADD integer TO identifier

***Input:***
- INPUT identifier1; identifier2; … identifierN.
- Takes in values to be held in identifiers

***Output:***
- PRINT “stuff to print”; identifier1.
- Any combination of strings/identifiers
